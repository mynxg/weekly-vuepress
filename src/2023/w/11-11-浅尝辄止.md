> Draw-a-UI ：让你像画画一样设计网页。 你只需在草图软件里随手画出你想要的网页界面，如简单的按钮或文本框，基本线框图。 然后它会使用 tldraw 和 GPT-4 Vision API 来根据你绘制的线框图生成 HTML页面。 主要原理是将当前的画布 SVG 转换为 PNG 图像，然后将该图像发送给 GPT-4 Vision，并指示其返回一个包含 Tailwind CSS 的单个 HTML 文件。
