                                              电子书籍下载

                                              1、鸠摩搜书：http://jiumodiary.comopen in new window 这个还是能找到不少书籍的

                                              2、无名图书：http://book123.infoopen in new window 电子书搜索下载网站，有上万本书籍，提供pdf、epub等格式，免费、无需注册登录，直接就能下载

                                              3、莫若书单：http://mrsd.topopen in new window 以pdf、epub格式为主，数量丰富，可以直接下载

                                              4、Project Gutenberg：http://gutenberg.orgopen in new window 收集了超过70000本电子书，所有书籍均为志愿者分享

                                              5、ManyBooks：http://manybooks.netopen in new window 收集了50000多本电子书，并提供多种格式供用户下载

                                              6、Forgotten Books：http://forgottenbooks.comopen in new window 目前已有129多万册外文书籍，涵盖多个领域，可在线阅读或下载

                                              7、PDFDrive：http://pdfdrive.comopen in new window 提供8400多万本电子书籍，其中有不少中文书籍，无下载限制

                                              8、Planet eBook：http://forgottenbooks.comopen in new window 主要提供古典文学著作，可以在线阅读或下载

                                              9、Loyal Books：http://loyalbooks.comopen in new window 提供了7000多书籍供用户下载，包括有声书和电子书！主要有儿童读物、小说、诗歌、历史等！都是免费的

                                              10、Freebookcentre：http://freebookcentre.netopen in new window 免费下载数千本技术类书籍，包括计算机科学、编程语言、物理学、医学、化学、数学、商业金融等种类

                                              11、搬书匠：http://banshujiang.cnopen in new window 提供编程开发类书籍，免费下载

                                              12、SaltTiger：http://salttiger.comopen in new window 编程类书籍下载网站，数量挺丰富
